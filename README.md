### What is OpenVPN Event Viewer?###

OpenVPN stores logs in text files which is very hard to read. 

OpenVPN Event Viewer offers a human readable format of the open vpn connection logs.

### Tech Stack ###

* Asp.NET MVC
* Asp.NET Web API
* RavenDB
* Python
* jQuery
* mustache.js
* Highcharts
